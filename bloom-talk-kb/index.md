# 胎教研究知识库索引

本目录由《各类胎教类型的开始孕周、实施细节与证据支持综述》拆分整理而成，已去除原始文档中的引用占位、图片占位等无效信息，保留正文内容并统一为一致的 Markdown 结构。

`summary.md` 承接总览、分类框架、发育时间窗、综合对比、风险管理与研究展望；其余文件按具体胎教类型拆分。理论上，`summary.md` 与九个专题文件加总后，对应完整调研报告的正文信息。

## 文件清单

- [summary.md](./summary.md)：总览、定义与分类框架、发育时间窗、综合对比表、时间线、风险管理、证据缺口。
- [nutrition.md](./nutrition.md)：营养胎教。
- [exercise.md](./exercise.md)：运动胎教。
- [emotion.md](./emotion.md)：意念胎教。
- [dialog.md](./dialog.md)：对话胎教。
- [music.md](./music.md)：音乐胎教。
- [touch.md](./touch.md)：抚摸胎教。
- [light.md](./light.md)：光照胎教。
- [art.md](./art.md)：美育胎教。
- [nature.md](./nature.md)：自然胎教。

## 使用顺序建议

1. 先阅读 [summary.md](./summary.md) 了解整体框架与风险边界。
2. 再按主题进入对应文件查看完整细节。
